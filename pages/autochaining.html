Auto Chaining

   An automated gameplay program developed using Node.js, leveraging libraries like Robot.js and Tesseract.js. The program uses Robot.js to simulate mouse and keyboard actions, enabling automated interactions within a game. Tesseract.js is employed for Optical Character Recognition (OCR) to analyze on-screen text and make decisions based on real-time game data. This project demonstrates a combination of automation and text recognition to enhance efficiency and performance in gameplay tasks.

   Chaining is killing the same monster repeatedly. The goal of chaining is to find the best monster, which consists of 6 stars, a secret trait, and a special color (misprint) . This program will help us kill the same monster repeatedly until we find the best one.
